## docker shortcuts 

quick and dirty way to autocomplete docker usual commands in a bash terminal


- dkr-shell
- dkr-images
- dkr-ps
- dkr-cleanc
    - clean up dead containers
- dkr-cleani
    - clean up untag / named images

## how to install

`curl -s  https://raw.githubusercontent.com/2mia/dkr/master/install.sh | bash`
